## 3.Kafka

### 3.1 安装

* **下载**

  下载Scala 2.12的Kafka 2.3.0版本。

  ```bash
  $ wget https://archive.apache.org/dist/kafka/2.3.0/kafka_2.12-2.3.0.tgz
  ```

* **解压**

  ```bash
  $ tar -zxvf kafka_2.12-2.3.0.tgz
  ```

* **移动到软件目录**

  比如`/root/app/kafka`

  ```bash
  $ mkdir /root/apps/kafka/
  $ mv kafka_2.12-2.3.0 /root/apps/kafka/
  ```
